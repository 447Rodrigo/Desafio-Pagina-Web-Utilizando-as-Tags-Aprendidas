Seção com Âncora

<h1> até <h6>: São cabeçalhos de diferentes níveis de importância, onde <h1> é o mais importante e <h6> o menos importante. Eles são usados para estruturar o conteúdo do seu site.

<hr>: Cria uma linha horizontal para separar seções de conteúdo.

Parágrafo de texto: <p> Define um parágrafo, onde é usado para separar blocos de texto.

Texto marcado: <mark> Destaca um texto dentro de um parágrafo.

Texto pequeno: <small> Define um texto com um tamanho de fonte menor, geralmente usado para notas de rodapé ou informações menos relevantes.

Itálico: <i> Define um texto como itálico, usado para destacar palavras ou frases.

Sublinhado: <u> Define um texto sublinhado, geralmente usado para indicar links.

Negrito: <strong> Define um texto importante ou enfatizado, geralmente exibido em negrito.

<ol> Cria uma lista ordenada (com números ou letras) de itens.

  1. Item 1
  2. Item 2
  3. Item 3

<ul> Cria uma lista não ordenada (com marcadores).>

  • Item A
  • Item B
  • Item C

<li> Define um item de lista, seja em uma lista ordenada ou não ordenada.

Link para Exemplo <a> Define um hiperlink para outra página ou recurso na web.

Texto Subscrito: <sub> Define um texto como subscrito, geralmente usado para fórmulas químicas ou notas de rodapé.

Texto Sobrescrito : <sup> Define um texto como sobrescrito, geralmente usado para expoentes ou notas de rodapé.

<blockquote> Define uma citação em bloco, geralmente com recuo, para destacar citações ou trechos importantes.

Citação importante

Texto deletado: <del> Define um texto que foi deletado ou removido, geralmente exibido com uma linha através dele.

Define uma abreviação ou acrônimo, e você pode adicionar uma explicação quando o mouse passa sobre o texto.

HTML

A âncora é criada usando o atributo id="ancora". Em seguida, criamos um link para essa âncora usando <a href="#ancora">Ir para a Seção com Âncora</a>. Quando clicamos nesse link, a página rola automaticamente até a seção com a âncora. Isso é especialmente útil em páginas longas para navegação interna.
